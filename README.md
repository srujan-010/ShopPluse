# ShopPulse - Smart Shop Management

ShopPulse is a modern full-stack web application for shop owners to manage multiple shops, track stock, sales, and generate detailed reports.

## Tech Stack
- **Frontend**: React, Vite, Lucide Icons, Framer Motion, Chart.js
- **Backend**: Node.js, Express, Mongoose, JWT
- **Database**: MongoDB Atlas

## Getting Started

### 1. Prerequisites
- Node.js installed
- MongoDB connection string (already configured in `server/.env`)

### 2. Setup Backend
```bash
cd server
npm install
npm run start (or node index.js)
```

### 3. Setup Frontend
```bash
cd client
npm install
npm run dev
```

## Features
- **Owner Dashboard**: Real-time business metrics and charts.
- **Shop Management**: Handle multiple business locations.
- **Inventory Tracking**: Manage products and receive low-stock alerts.
- **Sales Registry**: Log daily sales with auto-profit calculation.
- **Reporting**: Export reports to PDF and Excel.
